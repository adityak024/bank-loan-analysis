# 🏦 Bank Loan Analysis

## 📌 Project Overview
Performed EDA on 38,576 bank loan records to uncover trends in loan 
applications, funded amounts, repayment behavior, and risk profiling.

## 📊 Key Insights
- Total Loan Applications: 38,576
- Total Funded Amount: $435.76M
- Good Loan %: 86.18% | Bad Loan %: 13.82%
- Avg Interest Rate: 12.05% | Avg DTI: 13.33%

## 🛠 Tools & Libraries
Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly

## 📈 Analysis Covers
- Monthly trends (funded, received, applications)
- Regional analysis by US state
- Loan term breakdown (36 vs 60 months)
- Employment length impact
- Loan purpose distribution
- Home ownership segmentation
